# SecurityTool

MuLuBlast.py 			 用于爆破网站目录

SQLii.py						SQL注入的简单脚本

WebsiteThread.py      多线程爬取网站的链接

WhoisQuery.py           用于whois批量查询

DnsQuery.py			   DNS爆破

```
python DnsQuery.py 域名 -t 线程数 -f 文件名
命令行参数： 
	-t 线程数
	-f 传入爆破字典文件
	-o 扫描内容输出的文件
```
